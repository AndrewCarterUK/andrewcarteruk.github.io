A Case Study

How Anthony Ferrara Hacked StackOverflow

(Accidently)

Anthony was working for a company with a firewall

The firewall stripped certain headers

This broke many sites

He set up a proxy server, and browsed through that

One day an 'Admin' button appeared on StackOverflow

Developer Console...

Database Query Interface...

Admin Chat...

How?

His proxy server added an 'X-Forwarded-For' header

Anthony used an SSH tunnel to connect to his proxy

In the eyes of his proxy server, his IP address was...

127.0.0.1

StackOverflow trusted the X-Forwarded-For header

Examples from PHP

Slim 3.0.0 Beta and RC1


public function getIp()
{
    if ($this->hasHeader('X-Forwarded-For')) {
        $forwardedFor = $this->getHeaderLine('X-Forwarded-For');
        return trim(current(explode(',', $forwardedFor)));
    }

    return
        isset($this->serverParams['REMOTE_ADDR']) ?
        $this->serverParams['REMOTE_ADDR']        :
        null
    ;
}

protected function validateSignature($gitHubSignatureHeader, $payload)
{
    list ($algo, $gitHubSignature) = explode("=", $gitHubSignatureHeader);
    $payloadHash = hash_hmac($algo, $payload, $this->secret);
    return ($payloadHash == $gitHubSignature);
}
